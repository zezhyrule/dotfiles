Umwandlung von Text in gesprochene Sprache in Clearly

Setzen Sie sich zurück und lassen Sie die Blogs, Artikel und Webseiten von Clearly vorlesen. Dank der Umwandlung von Text in gesprochene Sprache (Sprachwiedergabe) können Evernote Premium-Abonnenten nun von diesem Feature profitieren.

Funktionsweise

Klicken Sie in der Seitenleiste von Clearly auf das Symbol für Sprachwiedergabe. Clearly erkennt die Sprache des Artikels automatisch und liest den Text laut vor.

Nur Teile wiedergeben

Möchten Sie nur bestimmte Teile eines Artikels in Sprache umwandeln lassen? Mit den Schaltflächen der Seitenleiste können Sie den nächsten bzw. vorherigen Satz ansteuern oder die Wiedergabe anhalten.

Mitlesen

Clearly markiert den aktuell vorgelesenen Text zum Mitlesen. Damit können Sie bei der Sprachwiedergabe andere Clearly-Funktionen wie Markieren und Ausschneiden nutzen.
